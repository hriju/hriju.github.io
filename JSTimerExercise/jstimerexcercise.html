4. What is "Unobtrusive Javascript"? What is the practical application of Unobtrusive Javascript (and the reasons for using it)?

Ans: Is a way to write javascript to separate document content and script content. If JavaScript features are partially or fully absent in a user's web browser, then the user notices as little as possible any lack of the web page's JavaScript functionality. This approach is useful in so many ways as it makes our code less error prone, easy to update and to debug.

Practical example is we can write three separate file for HTML , CSS and behavior page(JavaScript) then it will be called Unobtrusive way of Javascript code writing.
