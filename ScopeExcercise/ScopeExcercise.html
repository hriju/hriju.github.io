2. What is the difference between a method and function?

Answer: In way we can say that Functions and methods are the same in javascript, but a method is a function, which is a property of an object. A function can be called directly by its name but methods are called by its object name and its method name using dot notation or square bracket notation. Function has access to the data passed to that function, but a method has access to the properties of the class it belongs to.

10. Write a Javascript function that takes any number of input arguments and returns the product of the arguments. 

  function productOfArguments() {
    let i;
    let productResult = 1;
    for (i = 0; i < arguments.length; ++i) {
      productResult *= arguments[i];
    }
    return productResult;
};
